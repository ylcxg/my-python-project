# 电商用户行为分析项目

## 项目简介
基于淘宝用户行为数据集，使用 Python 进行数据清洗、探索性分析和可视化，旨在洞察用户行为模式。

## 项目内容
1.  **数据清洗**：处理缺失值，转换时间戳格式。
2.  **热门商品分析**：统计并可视化被浏览次数最多的 Top 10 商品。
3.  **转化漏斗分析**：对比浏览量和购买量，量化用户流失，计算转化率。

## 主要结论
*   在 10 万条样本数据中，用户浏览行为占绝大多数，购买转化率约为 2.2%。
*   商品 ID 为 `3027414` 的商品是浏览量最高的热门商品。
*   从浏览到购买的转化漏斗存在显著的用户流失，建议优化商品详情页或增加促销引导。
